🚀 **OpenClaw Memory v0.4.0 已发布**

这次重点做了「防阻塞 + 稳定检索 + 可审计」三件事：

1) **记忆注入预算上线**  
- 新增每文件/总量双阈值，防止上下文无限膨胀  
- 目标：减少长时间“无响应”与超时

2) **动态记忆注入（按场景最小化）**  
- 支持 `main / ops / btc / quant` 四类 profile  
- 按优先级自动裁剪上下文，只带必要信息

3) **记忆冲突检测**  
- 长期记忆写入前先做规则/路由冲突扫描  
- 避免历史规则互相打架

4) **检索健康 watchdog**  
- 每 30 分钟巡检一次记忆检索健康  
- 连续异常才确认告警，降低噪声误报

5) **夜间 QMD 自动维护**  
- 每天夜间自动 `qmd update`  
- 仅在 backlog 超阈值时才 `embed`，兼顾效果与成本

6) **文档与安装流程同步升级**  
- README（中英文）、架构文档、操作手册、AI bootstrap 提示词已更新  
- 安装后核验步骤更完整，开箱可用性更强
